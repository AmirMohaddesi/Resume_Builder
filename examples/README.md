# Examples

This directory contains example files for testing and reference.

## Contents

- **`expected_profile.json`** - Example profile structure
- **`profile_skills.json`** - Example skills data
- **`resume_full_output.json`** - Example full resume output structure

## Usage

These files are provided as reference examples. They are not used by the application during normal operation.

For actual profile data, use:
- `src/resume_builder/data/profile.json` (user-specific, gitignored)
- `output/user_profile.json` (generated from uploaded resume)
